Desafio de Projeto - Tags HTML

Este website demonstra o uso de diversas tags HTML aprendidas durante o curso

Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem fundamental para criação de páginas web. Neste projeto, exploraremos diversas tags HTML e suas aplicações práticas.

O objetivo principal é demonstrar o uso correto das tags solicitadas no desafio, criando uma página semântica e bem estruturada.

Nota: Este é um projeto demonstrativo para fins educacionais.

Cabeçalhos (Headings)

Os cabeçalhos definem a hierarquia e a estrutura do conteúdo. Eles vão de <h1> (mais importante) até <h6> (menos importante).

Listas

Listas são essenciais para organizar informações. HTML suporta listas ordenadas (<ol>) e não ordenadas (<ul>).

Lista Ordenada (ol)

  1. Primeiro item da lista
  2. Segundo item com subitens:
    1. Subitem A
    2. Subitem B
  3. Terceiro item

Lista Não Ordenada (ul)

  • Item de lista
  • Outro item com subitens:
    • Subitem 1
    • Subitem 2
  • Último item

Links e Navegação

A tag <a> é usada para criar hyperlinks, a base da navegação web.

Exemplos de links:

  • Link para a introdução (âncora na mesma página)
  • Link externo (abre em nova aba)
  • Link de e-mail
